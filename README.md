## ðŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for local development)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio-website.git
```

2. Navigate to the project directory:
```bash
cd ITE03_FinalProject
```

3. Open `index.html` in your web browser:
   - Simply double-click the file, or
   - Use a local server (e.g., `python -m http.server` or `npx serve`)

### Local Development

For the best development experience, use a local server:

**Using Python:**
```bash
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

**Using Node.js (npx):**
```bash
npx serve
```

## ðŸŽ¨ Customization

### Changing Colors

Edit the CSS variables in `style.css`:

```css
:root {
    --pink: #ff80b3;        /* Primary color */
    --dark: #1a1a1a;        /* Text color */
    --gray: #555;           /* Secondary text */
    --light-bg: #fff1f7;    /* Background color */
    --white: #ffffff;       /* White */
}
```

### Adding New Sections

1. Add HTML structure in `index.html`
2. Add corresponding CSS in `style.css`
3. Update navigation links if needed
